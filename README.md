# 💻 Heizel Is Hacked

A fake hacking-themed birthday website that turns memories, jokes, and moments into an interactive digital experience.

## 🚨 Overview

**Heizel Is Hacked** is a playful web project designed to look like a real computer breach. Visitors are greeted with terminal-style animations, system scans, and "classified" files that gradually reveal photos, memories, and personalized messages.

The project combines storytelling, web design, and visual effects to create a memorable surprise that feels like a hacker movie before transforming into a heartfelt tribute.

## ✨ Features

* Terminal-inspired interface
* Fake hacking animations
* Interactive file reveals
* Personalized memories and messages
* Responsive design
* Glitch effects and cinematic transitions
* Immersive storytelling experience

## 🎯 Purpose

This project was created as a creative digital gift to celebrate a friend in a unique way. Instead of a traditional birthday card or slideshow, it presents memories through an engaging narrative that keeps visitors curious about what will be uncovered next.

## ⚠️ Disclaimer

This website is a fictional experience created for entertainment purposes only.

No real hacking, data collection, security intrusion, or unauthorized access occurs at any point. All effects, terminal outputs, and system messages are simulated.

## 📸 Inspiration

Inspired by hacker movie interfaces, terminal aesthetics, digital storytelling, and the idea that memories can be revealed like hidden files waiting to be discovered.
